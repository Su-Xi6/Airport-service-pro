# 后台管理端样式规范

## 1. 侧边栏
- 背景色：深色/暗色 (#1a1a2e 或 #2c3e50)
- 文字颜色：白色或浅色 (#ecf0f1)
- 选中项高亮色：略浅于背景色 (#34495e)

## 2. 主内容区
- 背景色：浅灰 (#f5f6fa)
- 卡片背景：白色 (#ffffff)
- 卡片圆角：8px
- 卡片阴影：0 2px 8px rgba(0,0,0,0.1)

## 3. 表格
- 表头背景：深色 (#2c3e50)
- 表头文字：白色
- 表格圆角：8px
- 行间隔色：奇数行白色，偶数行浅灰 (#f9f9f9)
- 边框：1px solid #e0e0e0

## 4. 按钮
- 主按钮：蓝色 (#3498db)，圆角 6px
- 危险按钮：红色 (#e74c3c)，圆角 6px

## 5. 字体
- 系统默认字体栈：-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
